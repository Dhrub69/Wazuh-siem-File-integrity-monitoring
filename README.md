# 🛡️Wazuh-siem-File-integrity-monitoring

#### Cybersecurity homelab using Wazuh SIEM + XDR with Ubuntu (Manager) and Windows (Agent) in VirtualBox. Includes File Integrity Monitoring (FIM), agent setup, and dashboard monitoring.

## 🛡️ Wazuh SIEM + XDR Cybersecurity lab

## 📌 Overview
#### This project demonstrates a full cybersecurity lab using Wazuh SIEM + XDR solution deployed in a virtual environment using VirtualBox. It includes Ubuntu as the Wazuh Manager and Windows as the Wazuh Agent.

## 🧱 Lab Architecture
#### - Ubuntu VM → Wazuh Manager + Dashboard + Indexer
#### - Windows VM → Wazuh Agent
#### - VirtualBox → Network: Host-only / NAT + Host-only

## ⚙️ Features Implemented
#### - Wazuh Manager installation (Ubuntu)
#### - Wazuh Agent installation (Windows)
#### - Agent enrollment and key authentication
#### - File Integrity Monitoring (FIM)
#### - Real-time event monitoring
#### - Security alerts visualization in dashboard

## 🔍 File Integrity Monitoring Demo
#### - Monitored system folders on Windows agent
#### - Detected file changes via Wazuh dashboard alerts
